    function esql_home_url_watch(){
            let el          = document.querySelector('.js-home-page-url-watch');
            if(!el) return
            let rect        = el.getBoundingClientRect();
            let top         = rect.top;
            let breakpoint  = window.innerHeight / 2;
            let current_url = window.location.pathname;
            let article_url = document.querySelector('.esql-article').dataset.url;
            let home_url    = "/"

            let IS_ARTICLE_VISIBLE = true;
            let IS_HOME_VISIBLE    = false;
            let IS_ARTICLE_URL     = true;
            let IS_HOME_URL        = false

            if( current_url == article_url ) {
                IS_ARTICLE_URL = true;
                IS_HOME_URL = false;
            }
            else if( current_url == home_url ) {
                IS_ARTICLE_URL = false;
                IS_HOME_URL = true;
            }

            if( top > breakpoint ){
                IS_ARTICLE_VISIBLE = true;
                IS_HOME_VISIBLE = false;
            }
            else {
                IS_ARTICLE_VISIBLE = false;
                IS_HOME_VISIBLE = true;
            }

            if(! app || !app.__vue__)
                return;

            if( IS_ARTICLE_VISIBLE && IS_HOME_URL ) {
                window.history.pushState(null, null, article_url);
                app.__vue__.$data.isHomePageVisible = false;
            }
            else if( IS_HOME_VISIBLE && IS_ARTICLE_URL ) {
                window.history.pushState(null, null, home_url);
                app.__vue__.$data.isHomePageVisible = true;
            }
    }

    function esql_recommended_articles_watch(){
        let article_content = document.querySelectorAll('.article-content')[0]
        let article_content_rect = article_content.getBoundingClientRect()
        let article_content_top  = article_content_rect.top;

        try{
            if( article_content_top < 0 ) {
                app.__vue__.canRecommendedArticlesBeVisible = true;
            }
            else {
                app.__vue__.canRecommendedArticlesBeVisible = false;
            }
        }
        catch(e){}
    }

    document.addEventListener('scroll', ()=>{

        esql_recommended_articles_watch();
        esql_home_url_watch();

    });

                        <p><strong>W związku z premierą nowych modeli Alfy Romeo, robimy przegląd ulubionych usportowionych wersji aut. Oto nasza subiektywna lista!</strong></p>
<p><strong>Alfa Romeo Giulia GTA i GTAm</strong></p>
<p>Mówi się, że nowe Alfy to legenda z technologią zaczerpnięta z Formuły 1. Dlaczego? Modele GTA i GTAm zostały oparte na flagowym modelu marki, czyli Giulii Quadrifoglio. Pierwszy z nich to tzw. wersja cywilna - drugi natomiast jest przeznaczony do jazdy po torze, dlatego otrzymał dwa kubełkowe fotele, klatkę bezpieczeństwa i sześciopunktowe pasy.</p>
<p>Co zmienili projektanci w samej sylwetce? Przede wszystkim zmniejszyła się waga - nowa Giulia ważny aż o 100 kg mniej niż jej poprzedniczka. Udało się to osiągnąć za sprawą stworzenia wielu elementów z poliwęglanu, który jest trwały i bardzo lekki. Dodatkowo, jak na rasowe sportowe auto przystało, dodano dyfuzor i spojler, które poprawiają aerodynamikę i dodają wyglądowi pazura. Natomiast pod maską obu wersji kryje się benzynowy silnik 2.9 V6 Bi-turbo, który generuje 540 KM. Dzięki niemu osiągnięcie setki zajmuje zaledwie 3,6 sekundy.</p>
<figure id="attachment_25815" aria-describedby="caption-attachment-25815" style="width: 5315px" class="wp-caption alignnone"><a class="gallery" href="https://sznyt.pl/wp-content/uploads/2020/03/18.jpg"><img class="wp-image-25815 size-full" src="https://sznyt.pl/wp-content/uploads/2020/03/18.jpg" alt="" width="5315" height="3543" /></a><figcaption id="caption-attachment-25815" class="wp-caption-text">fot. Alfa Romeo</figcaption></figure>
<p>Jeśli macie trochę odłożonej gotówki i chcielibyście kupić sobie usportowioną Giulię to musicie się spieszyć - Alfa Romeo wypuszcza na rynek zaledwie po 500 sztuk każdej wersji. Każdy kupujący GTA lub GTAm zostaje automatycznie ambasadorem marki i otrzymuje unikatowy sportowy strój oraz kask. Dodatkowo ma możliwość odbycia specjalnego kursu z Alfa Romeo Driving Academy.</p>
<figure id="attachment_25816" aria-describedby="caption-attachment-25816" style="width: 1920px" class="wp-caption alignnone"><a class="gallery" href="https://sznyt.pl/wp-content/uploads/2020/03/alfa-romeo-giulia-gta-7.jpg"><img class="wp-image-25816 size-full" src="https://sznyt.pl/wp-content/uploads/2020/03/alfa-romeo-giulia-gta-7.jpg" alt="" width="1920" height="1080" /></a><figcaption id="caption-attachment-25816" class="wp-caption-text">fot. Alfa Romeo</figcaption></figure>
<p><strong>Toyota GR Yaris</strong></p>
<p>Czekamy na tę premierę z niecierpliwością - Toyota ujawniła w końcu wygląd Yarisa GR i jesteśmy zaintrygowaniu nowym usportowionym modelem japońskiego koncernu. Jak zaznacza producent, Yaris GR powstał na bazie doświadczeń fabrycznego zespołu WRC, a o ścisłych wytycznych decydował sam Tommy Mäkinen - kierownik teamu Gazoo Racing. Nic więc dziwnego, że w tym małym sportowym aucie zastosowano najbardziej zaawansowane technicznie rozwiązania, które z pewnością sprawdzą się do jazdy na codzień i do zabawy na torze.</p>
<p>Co jeszcze wiemy o hiperhatchu Toyoty? Model będzie miał silnik 1.6 turbo (261 KM) i będzie ważył 1280 kg. Niewiadomą pozostaje czas przyspieszenia 0-100 km/h, ale Toyota na razie podaje, że będzie to „mniej niż 5,5 s”.</p>
<figure id="attachment_25844" aria-describedby="caption-attachment-25844" style="width: 1680px" class="wp-caption alignnone"><a class="gallery" href="https://sznyt.pl/wp-content/uploads/2020/03/11111_tcm-3031-1836822.jpg"><img class="wp-image-25844 size-full" src="https://sznyt.pl/wp-content/uploads/2020/03/11111_tcm-3031-1836822.jpg" alt="" width="1680" height="1021" /></a><figcaption id="caption-attachment-25844" class="wp-caption-text">fot. Toyota</figcaption></figure>
<figure id="attachment_25845" aria-describedby="caption-attachment-25845" style="width: 1600px" class="wp-caption alignnone"><a class="gallery" href="https://sznyt.pl/wp-content/uploads/2020/03/home_header_tcm-1015-1838303.jpg"><img class="wp-image-25845 size-full" src="https://sznyt.pl/wp-content/uploads/2020/03/home_header_tcm-1015-1838303.jpg" alt="" width="1600" height="964" /></a><figcaption id="caption-attachment-25845" class="wp-caption-text">fot. Toyota</figcaption></figure>
<p><strong>Renault Megane RS Trophy</strong></p>
<p>Małe a jakie zwinne i szybkie! Renault Megane RS w standardowej odmianie ma silnik 1,8 turbo o mocy 280 km - w wersji Trophy zwiększono moc do 300 km. Auto potrzebuje zaledwie 5,7 s aby rozpędzić się od 0 do 100 km/h (o 0,1 s szybciej), a prędkość maksymalna wynosi 260 km/h (o 5 km/h).</p>
<p>Jeśli chodzi o moment obrotowy to w przypadku skrzyni manualnej wynosi on 400 Nm, ale jeśli zdecydujemy się na 2-sprzegłową skrzynię automatyczną EDC jest to 420 Nm.</p>
<figure id="attachment_25818" aria-describedby="caption-attachment-25818" style="width: 1324px" class="wp-caption alignnone"><a class="gallery" href="https://sznyt.pl/wp-content/uploads/2020/03/zzz-6.jpg"><img class="wp-image-25818 size-full" src="https://sznyt.pl/wp-content/uploads/2020/03/zzz-6.jpg" alt="" width="1324" height="745" /></a><figcaption id="caption-attachment-25818" class="wp-caption-text">fot. <figure id="attachment_25819" aria-describedby="caption-attachment-25819" style="width: 972px" class="wp-caption alignnone"><a class="gallery" href="https://sznyt.pl/wp-content/uploads/2020/03/zzz-4.jpg"><img class="wp-image-25819 size-full" src="https://sznyt.pl/wp-content/uploads/2020/03/zzz-4.jpg" alt="" width="972" height="547" /></a><figcaption id="caption-attachment-25819" class="wp-caption-text">fot. Renault</figcaption></figure>
<p><strong>Škoda Octavia RS 245</strong></p>
<p>Praktyczny i dynamiczny - j<a href="https://sznyt.pl/2018/02/16/skoda-octavia-rs-245-najmocniejsza/" target="_blank" rel="noopener">akiś czas temu mieliśmy przyjemność przetestować nowy model Octavii</a>, który swoją stylistyką znacząco odróżnia się od swoich poprzedników. Design jest odważny, agresywny i nowoczesny. Niewiele już w Octavii zostało ciężkości i siermiężności, którą pamiętamy z modeli produkowanych jako pierwsza generacja w końcu lat 90-tych XX wieku i nam się to bardzo podoba. W nasze ręce wpadł egzemplarz w praktycznej wersji nadwozia liftback, ale jeśli 590-litrowy bagażnik jest dla Was za mały, Octavia RS 245 dostępna jest również jako combi (610 l). Obie wersję osiągają prędkość maksymalną 250 km/h i rozpędzają się do 100 km/h w czasie - odpowiednio, 6,6 i 6,7 sekundy.</p>
<p>Oprócz wyglądu od "standardowej" wersji RS oczywiście odróżnia ją również zwiększona moc silnika (bazowo 230 KM) oraz specjalny pakiet stylistyczny, w którego skład wchodzą lakierowane na czarno 19-calowe felgi, końcówki wydechów, zestaw listw oraz coś, co specjalnie ucieszy fanów prawdziwie sportowej jazdy - blokada mechanizmu różnicowego.</p>

Škoda</figcaption></figure>
<figure id="attachment_25843" aria-describedby="caption-attachment-25843" style="width: 1200px" class="wp-caption alignnone"><a class="gallery" href="https://sznyt.pl/wp-content/uploads/2020/03/octavia-rs-245-2.1200.jpg"><img class="wp-image-25843 size-full" src="https://sznyt.pl/wp-content/uploads/2020/03/octavia-rs-245-2.1200.jpg" alt="" width="1200" height="675" /></a><figcaption id="caption-attachment-25843" class="wp-caption-text">fot. Škoda</figcaption></figure>
<p><strong><br />
Ford Focus RS III</strong></p>
<p>W 2015 roku Ford zaprezentował nową odmianę RS, która powstała na bazie III generacji Focusa. Z pewnością to najciekawsza sportowa odmiana tego modelu, który zyskał już całą rzeszę fanów na świecie. Auto napędzane jest 2,3 litrowym, turbodoładowanym (twin-scroll) silnikiem Ecoboost o mocy 350 KM i ma system napędu 4x4 Ford Performance All-Wheel-Drive. Z takim napędem osiągniemy setkę w 4,7 sekundy.</p>
<p>Według zapowiedzi Ford szykuje premierę swojego nowego modelu RS, która miałaby się odbyć w drugiej połowie 2020 roku. Jeśli wierzyć plotkom, pod maską znajdzie się ten sam silnik 2.3 R4, ale połączony z silnikiem elektrycznym zdolnym napędzać tylko tylną oś (łącznie ponad 400 KM). Będzie to innowacyjny system eTwinster, nad którym pracuje firma GKN Automotive.</p>
<figure id="attachment_25828" aria-describedby="caption-attachment-25828" style="width: 960px" class="wp-caption alignnone"><a class="gallery" href="https://sznyt.pl/wp-content/uploads/2020/03/Ford-Focus-RS-III-2015_04.jpg"><img class="size-full wp-image-25828" src="https://sznyt.pl/wp-content/uploads/2020/03/Ford-Focus-RS-III-2015_04.jpg" alt="" width="960" height="540" /></a><figcaption id="caption-attachment-25828" class="wp-caption-text">fot. Ford Focus</figcaption></figure>
<figure id="attachment_25829" aria-describedby="caption-attachment-25829" style="width: 960px" class="wp-caption alignnone"><a class="gallery" href="https://sznyt.pl/wp-content/uploads/2020/03/3U1A7910.jpg"><img class="size-full wp-image-25829" src="https://sznyt.pl/wp-content/uploads/2020/03/3U1A7910.jpg" alt="" width="960" height="540" /></a><figcaption id="caption-attachment-25829" class="wp-caption-text">fot. Ford Focus</figcaption></figure>
<p><strong>Seat Leon Cupra</strong></p>
<p>Cupra Leon w wariancie spalinowym ma odpowiednio 300 i 310 KM, natomiast jako hybryda – 245 KM. Ta najmocniejsza opcja dostępna jedynie w wersji Sports Tourer, czyli jako kombi. Podstawowym wariantem jest jednak hybryda oparta na silniku 1,4 TSI, który zapewnia 400 Nm maksymalnego momentu obrotowego. Na jednym naładowaniu przejedziemy autem około 60 km. Najmocniejsza wersja – czyli 310-konna Cupra Leon z silnikiem spalinowym – osiąga setkę w mniej niż 5 sekund. Co istotne, wszystkie warianty posiadają wyłącznie automatyczną przekładnię DSG.</p>
<p>Warta wspomnienia jest również unowocześniona linia nadwozia i wyrafinowane wykończenie nowego modelu: chromowane pedały, aluminiowe progi drzwi, elektrycznie otwierany dach panoramiczny, kubełkowe fotele. Dzięki tym detalom zupełnie inaczej patrzymy na dobrze nam znany model Seata.</p>

<p><strong>Mk8 Volkswagen Golf GTI 2020</strong></p>
<p>Zgadza się - Mk8 Volkswagen nie może być jeszcze naszym "ulubionym usportowionym modelem", ale nie ukrywamy, że pokładamy w tym modelu duże nadzieje. Dlaczego? Samochód, którego premiera ma się odbyć tego lata, będzie wyposażony w czterocylindrowy silnik z turbodoładowaniem, który przekazuje moc na przednie koła. Co ważne, ta moc będzie przekładana przez nowo opracowaną sześciobiegową manualną skrzynię biegów lub siedmiobiegowy automat DSG z manetkami łopatkowymi. Jak zapowiada producent, przyspieszenie do 100 km/h będzie w tym samochodzie osiągalne w zaledwie 6,4 sekundy.</p>
<p>Mamy nadzieję, że Volkswagen nie zawiedzie i nowy model faktycznie będzie jeszcze lepszy niż ostatni Golf GTI w wersji Performance Package.</p>
